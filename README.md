# tenkaTool v2+
tenkaFuma! Recruit confirm SR tool

텐카푸마 SR확정식 확인을 위한 어플리케이션

설치 후 실행시키면 Activity는 전부 꺼지고(백그라운드에서도) Service만 남음
Service에서 버튼 하나 생성

UI 설명

- 전원버튼
: 누르면 매크로 버튼 생성, 다시 누르면 매크로 끄고 매크로버튼 제거
- 검색버튼
: 실행시키면 처음 나타나는 버튼 드래그 가능, 드래그 할 때 나타나는 휴지통에 넣으면 프로그램 종료
: 드래그가 아니라 클릭할 시 화면스캔을 통해 전지역모집 태그확인 후 결과를 textBox에 출력 

- 휴지통
: 화면 하단 중앙에 위치. 검색버튼 드래그할 때만 보임
: 검색버튼을 끌어다 넣으면 프로그램 종료

- 텍스트박스
: 화면 상단에 위치. 첫 실행시 보이지 않음. 검색버튼을 누르면 결과를 보여주면서 나타남.
: 텍스트박스를 터치할 시 사라짐.

- 매크로버튼
: 누르면 1초간격으로 해당 위치 터치
